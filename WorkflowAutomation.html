About the Role

Omnera LLC is seeking a Workflow Automation Specialist to design and implement process automation between our website, vendors, logistics providers, and internal approval systems.

You will focus on event-based triggers, notifications, validation flows, and approval workflows.

What You’ll Design

  • Automated order notifications
  • Vendor confirmation workflows
  • Logistics validation alerts
  • Internal approval notifications
  • Payment release trigger workflows
  • Dashboard or reporting visibility

Key Responsibilities

Build automation flows using tools such as:

  • Zapier
  • Make (Integromat)
  • n8n
  • Low-code automation platforms
  • Configure event triggers based on API/webhook responses
  • Create real-time alerts (email, SMS, Slack)
  • Build conditional approval logic before payment release
  • Document workflows and process maps
  • Optimize automation reliability and error handling

Required Skills

  • 3+ years automation experience
  • Strong experience with workflow automation tools
  • Understanding of API triggers and webhook events
  • Experience building approval-based workflows
  • Strong documentation skills

Preferred

  • Experience with e-commerce platforms
  • Experience integrating payment triggers
  • Basic understanding of backend systems
