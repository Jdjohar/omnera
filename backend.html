Backend API & Systems Integration Developer

Location: Remote | Type: Contract or Part-Time (20–30 hrs/week)

Apply Now! Back to Home

Company

Omnera LLC

About the Role

Omnera LLC is seeking a skilled Backend API & Systems Integration Developer to build and manage integrations between our website, vendor partners, logistics providers, and payment systems.

You will design and implement the technical backbone of our automated order fulfillment and conditional payment workflow.

What You’ll Build

You will help automate a workflow where:

  • Customer places order on website
  • Order details automatically transmit to vendor
  • Logistics provider receives pickup request
  • Delivery company validates order
  • Internal notification confirms pickup validation
  • Payment to vendor is released after approval
  • Vendor ships the order

Key Responsibilities

  • Build secure REST API integrations between:
  • E-commerce platform
  • Vendor systems
  • Logistics/delivery providers
  • Payment gateway (Stripe or similar)
  • Implement webhook-based event handling
  • Design database structure for order lifecycle tracking
  • Develop transaction-based payment release logic
  • Implement audit logging and validation checkpoints
  • Ensure secure data handling and financial controls
  • Collaborate with automation specialist for trigger-based workflows

Required Skills

  • 4+ years backend development experience
  • Strong experience with:
  • REST APIs
  • Webhooks
  • Payment gateway integrations
  • Experience with:
  • Node.js / Python / Java (any backend stack)
  • PostgreSQL or MySQL
  • Familiarity with cloud platforms (AWS, Azure, or GCP)
  • Understanding of transactional systems and financial controls

Preferred

  • Experience integrating logistics APIs (UPS, FedEx, Shippo, etc.)
  • Experience building escrow-style or conditional payout systems
  • E-commerce workflow experience

Compensation

Competitive hourly or project-based compensation depending on experience.
